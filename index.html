Privacy Policy

As the development team of the OKU Chrome Extension, we highly value the privacy of our users. This privacy policy explains how our extension collects, uses, and protects user data.


Collected Data

The OKU extension only collects the following data related to user activity:

  • Click counts
  • Erroneous actions

This data is used solely to improve the performance of the extension and enhance the user experience.


Data Usage

The collected data is used only for analytical purposes. This data:

  • Is not sold or transferred to third parties.
  • Is not used for purposes other than the extension's intended functionality.
  • Is not used for credit evaluation or lending purposes.

Data Security

The security of user data is our priority. The data is protected against unauthorized access.

Free and Public Benefit
